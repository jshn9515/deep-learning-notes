# Code of Conduct

This project is a learning-oriented deep learning notes repository. The goal is to make deep learning concepts clearer, more intuitive, and easier to implement.

## Expected Behavior

- Be respectful and constructive.
- Point out mistakes with clear explanations when possible.
- Welcome questions from learners with different backgrounds.
- Discuss ideas, code, and explanations rather than attacking people.

## Unacceptable Behavior

- Personal attacks, harassment, or insulting language.
- Dismissive comments toward beginners.
- Spam or unrelated self-promotion.
- Repeated disruptive behavior in issues, discussions, or pull requests.

## Reporting

If you notice inappropriate behavior, you can contact the maintainer through GitHub issues or email if available.

The maintainer may remove comments, close discussions, or block users when necessary to keep the project welcoming and focused.
